# Purpose

Every Workspace begins with a Dashboard.

The Dashboard is not a reporting page.

The Dashboard is the operational command center.

Its purpose is to answer one question:

"What requires my attention right now?"

Everything else is secondary.

# Dashboard Rules

The dashboard shall never become a "wall of cards."

Information density must remain enterprise grade.

Maximum scrolling:

One screen height.

Everything critical visible immediately.
